# Unreal-Workshop-5.6
Interactive and Linear Storytelling Workshop Part 1 &amp; Part 2 for UE 5.6.1

This Workshop is for DAT590/DIT998 Interactive and Linear Storytelling  with Digital media Master's 7.5 credit Course.

It is broken up into two segments
- Cinematics
- Gameplay

Project Setup is the first release, you can then follow step by step with Workshop Instructions via a PowerPoint Presentation: [Unreal Workshops]()
